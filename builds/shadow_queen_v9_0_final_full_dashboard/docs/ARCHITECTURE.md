# Shadow Queen v9.0 Final Architecture

Final reference package for a peer-to-peer virtual DMV and distributed identity-governance simulator.

Core domains:
- identity
- credentials
- authority
- workflow
- notifications
- reporting
- persistence
- replication
- live mesh
- dashboard

Carbon users are external citizens/operators. Silicon users are internal agents/services. Both operate through identity, authority, workflow, audit, and evidence controls.
